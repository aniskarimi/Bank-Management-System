# Bank Management System (C++)

## Overview

This project is a simple **Bank Management System** developed in **C++** as the final project for the **Advanced Programming** course. It demonstrates the use of object-oriented programming (OOP) concepts such as inheritance, polymorphism, dynamic memory management, and abstraction.

The system allows users to manage customers and their bank accounts while supporting common banking operations.

## Features

* Add, edit, search, and delete customers
* Generate unique customer IDs automatically
* Support multiple bank accounts for each customer
* Three account types:

  * Current Account
  * Savings Account
  * Investment Account
* Automatic unique account number generation
* Deposit and withdraw money
* Record transaction history with date, amount, and transaction type
* Display customer information and all associated accounts
* Dynamic memory allocation for customers, accounts, and transaction history

## Object-Oriented Concepts

This project demonstrates the following C++ OOP concepts:

* Classes and Objects
* Encapsulation
* Inheritance
* Polymorphism
* Abstract Classes
* Virtual Functions
* Constructors and Destructors
* Dynamic Memory Allocation (`new` / `delete`)
* Pointers and Dynamic Arrays

1. Open the project in **Visual Studio** or any C++ IDE.
